### setb

#### 文法
```
 % setb HEX表記のアドレス HEX表記のデータ
```
指定したアドレスに指定したデータを書き込む

例
```
% setb 2 2f                                                                     
[0002H]<=2FH                                                                    
DONE                                                                            
```
